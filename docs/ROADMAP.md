# Roadmap — MÖBI Time (A1 Ponto)

Itens planejados para versões futuras. **Não implementados na V1.**

## Localização e presença

- Geolocalização no registro de ponto
- Geofence (cerca virtual por local de trabalho)
- Múltiplos locais / filiais

## Identificação e canais

- Reconhecimento facial
- Integração com WhatsApp (avisos e comprovantes)
- Modo quiosque (terminal fixo na empresa)
- Dashboard para TV (painel de presença em tempo real)

## Plataforma e integrações

- Multiempresa SaaS
- Integração com folha de pagamento
- API pública para integrações externas
